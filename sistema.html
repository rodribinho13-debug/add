Como o motor funciona

// 1. BUSCA RODADA ATUAL (API-Futebol) GET /v1/campeonatos/10/rodadas → acha rodada com status "andamento" ou "agendada" GET /v1/campeonatos/10/rodadas/{N} → lista as partidas // 2. BUSCA ESTATÍSTICAS DE CADA TIME GET /v1/campeonatos/10/times/{id}/estatisticas → divide por jogos → média por partida // 3. MODELO DIXON-COLES expGolsHome = (gols_marc_home_casa × gols_sofr_away_fora) / media_liga expGolsAway = (gols_marc_away_fora × gols_sofr_home_casa) / media_liga // 4. DISTRIBUIÇÃO POISSON P(home=i, away=j) = Poisson(expGolsHome,i) × Poisson(expGolsAway,j) P(Over 2.5) = ΣP(i+j > 2.5) P(BTTS) = ΣP(i>0 e j>0) P(HW) = ΣP(i>j) // 5. EV + KELLY FRACIONADO EV = prob×(odd−1) − (1−prob) Kelly = EV/(odd−1) × fracao_kelly × ajuste_simultaneas Valor = Kelly × patrimonio
